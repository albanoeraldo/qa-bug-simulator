# QA Bug Simulator

Simulador de bugs para treino de QA (reprodução, evidência e relatório).

## O que tem aqui
- 5 bugs simulados (duplicidade, UI não atualiza, loading infinito, erro intermitente, permissão)
- Logs com timestamp
- Botão de copiar relatório (pronto para colar em Jira/WhatsApp/Email)

## Como rodar
Abra o `index.html` no navegador.
